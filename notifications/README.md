Сервис ежедневных уведомлений для менеджеров.

### Возможности

- планировщик на APScheduler
- каждый будний день (пн–пт) в 8:00 по времени Москвы
- запрашивает у основного API эндпоинт `/internal/managers/new-responses`
- для каждого менеджера с ненулевым количеством новых откликов формирует письмо
- «отправляет» письмо, выводя текст в консоль

Формат письма:

> Здравствуйте, \[Имя]! У вас \[число] новых откликов, требующих обработки.

### Запуск локально

```bash
cd notifications
pip install -r requirements.txt
python main.py
```
